# 青岛推倒胡产品项目

本目录承载真实玩家产品“青岛推倒胡”的独立研发产物；不与 `mahjong-project-parse/` 中的既有源码解析产物混用。

当前处于研发基线的**阶段 0：范围与事实**。本阶段只做规则来源核验、分歧梳理和首局房间方案讨论，不改动玩法引擎，也不把未确认的地方规则当作产品定稿。

## 当前入口

- [规则讨论 V0](规则讨论-V0.md)：公开资料、规则分歧与首版候选方案。
- [场景锚点](scene-anchor.md)：首轮玩法讨论的目标和边界。
- [假设清单](assumptions.md)：尚未确认且会影响引擎的规则假设。
- [决策记录](debate-log.md)：已讨论的结论与待决问题。
- [当前状态](state.json)：本项目阶段、下一步和开放问题。

## 目录说明

- `evidence/`：外部规则来源和既有代码影响证据。
- `drafts/`：评审过程稿；当前尚未进入 PRD 撰写阶段。
- `output/`：仅存放确认后的正式 PRD、设计和验收交付物；当前为空。

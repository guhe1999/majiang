| 字段 | 内容 |
| :--- | :--- |

## 2. 页面概述

### 2.1 一句话定位

模块地图页把 `code-back/mahjong` 的入口、目录、文件族和职责整理成可扫描的结构图。

### 2.2 存在理由

1. 没有模块地图，lei 只能从文件名猜职责。
2. 没有源码引用，文档结论缺少可信来源。
3. 没有影响提示，he 无法判断新需求会动到哪里。

### 2.3 页面目标（可衡量）

| 目标维度 | 具体指标 |
| :--- | :--- |
| 结构理解 | 10 秒内看出入口、cmds、engine 三层关系 |
| 追溯性 | 每个核心模块至少有一个源码引用 |
| 影响判断 | 每个模块展示常见影响需求 |

### 2.4 不做什么

- 不逐行展示源码。
- 不做真实依赖分析图计算。
- 不修改源码目录。

## 3. 用户与场景

### 3.1 目标用户

| 维度 | 描述 |
| :--- | :--- |
| 角色 | lei 为主，he 辅助查看 |
| 技术熟练度 | 能理解 Go 文件和模块 |
| 使用设备 | 桌面浏览器 |
| 使用姿势 | 对照源码和文档阅读 |

### 3.2 核心使用场景

**场景 A：lei 学习源码结构**
- 时机：准备阅读源码。
- 前置：已拿到文件清单。
- 动作：查看模块树 → 展开文件族 → 复制源码路径。
- 情绪：希望快速建立全局结构。

**场景 B：he 评估需求影响**
- 时机：提出新需求前。
- 前置：知道需求大致类型。
- 动作：查看模块影响标签 → 跳转影响范围页。
- 情绪：希望判断风险大小。

### 3.3 用户核心诉求

1. 看懂模块职责。
2. 找到源码引用。
3. 判断模块影响范围。

## 4. 信息架构

### 4.1 区域划分

1. 模块树。
2. 模块详情。
3. 文件族分组。
4. 源码引用区。
5. 影响范围入口。

### 4.2 区域内容

**模块树**
- mahjong.go。
- cmds。
- engine。
- room / user / match / rpc / task / failover 文件族。

**模块详情**
- 职责摘要。
- 关键文件。
- 待验证点。

## 5. 界面内容

### 5.1 元素清单

**模块筛选**
- 选项：入口、命令、引擎、房间、用户、匹配、外部边界。
- 数据来源：模块索引。
- 显示条件：始终显示。

**源码引用**
- 内容：文件路径。
- 操作：复制路径。
- 显示条件：模块有来源文件时。

**影响标签**
- 内容：高影响 / 中影响 / 低影响。
- 数据来源：影响范围表。

### 5.2 辅助信息

- 文件数量。
- 行数统计。
- 解析状态。

## 6. 交互流程

### 6.1 页面加载与初始化

- 默认选中 engine 模块。
- 展示模块树和核心文件族。

### 6.2 核心操作流程

```text
选择模块 → 查看职责摘要 → 查看源码引用 → 跳转核心流程或影响范围
```

### 6.3 辅助操作

- 搜索文件名。
- 筛选文件族。
- 复制源码路径。

## 7. 状态与边界

### 7.1 页面状态全景

| 状态 | 触发条件 | 页面表现 |
| :--- | :--- | :--- |
| 初始 | 读取模块索引成功 | 展示模块树 |
| 输入 | 用户搜索文件 | 过滤模块树 |
| 成功 | 选择模块 | 展示详情 |
| 失败 | 模块无源码引用 | 显示未验证 |
| 异常 | 索引缺失 | 引导到脚本索引 |
| 空状态 | 尚未解析模块 | 展示文件清单入口 |

### 7.2 边界与容错

- 网络异常：不依赖网络。
- 会话过期：无登录态。
- 连续错误：保留错误原因。
- 异常输入：搜索无结果时展示推荐模块。
- 并发操作：文档冲突时以 Git 差异为准。

## 8. 业务规则

- 模块职责必须能回指源码路径。
- 推断性描述必须标记待验证。
- 模块影响标签不得替代真实代码评审。

## 9. 页面流转

```text
模块地图页
  ├─ 点击核心流程 → 核心流程页
  └─ 点击影响范围 → 影响范围页
```

## 10. 数据需求

| 数据 | 来源 | 用途 |
| :--- | :--- | :--- |
| 模块树 | JSON 索引 | 左侧结构 |
| 文件族 | 扫描脚本 | 分组展示 |
| 源码引用 | Markdown 文档 | 追溯 |

## 11. 验收标准

### 功能验收

- 展示至少 6 类模块分组。
- 每个核心模块有职责摘要。
- 支持源码路径复制。
- 能跳转核心流程页。
- 能跳转影响范围页。

### 体验验收

- 模块树可扫描。
- 文件路径使用等宽字体。
- 未验证状态清晰。
- 搜索结果明确。
- 移动端单列可读。

### 安全验收

- 不粘贴大段源码。
- 不修改源码。
- 未验证结论有标记。
- 脚本失败有提示。
- 不展示敏感数据。

## 2. 测试策略

| 优先级 | 测试维度 | 依据来源 | 理由 |
| :--- | :--- | :--- | :--- |
| P0 | 本地打开、首屏、无网络可读、脚本/同步基础耗时 | 蓝图 L8 | 确保工作台可用 |
| P1 | 搜索筛选、长表格、超长路径、页面切换 | 页面体验验收 | 防止规模稍大就卡顿 |

### 2.2 不测什么

- 不测 HTTP SLA、数据库查询、服务端并发、CDN 或真实麻将在线对局性能。

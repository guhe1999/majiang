# 页面清单

| 所属阶段 | 功能模块 | 页面名称 | 功能范围描述 |
| :--- | :--- | :--- | :--- |
| 阶段1(MVP) | 工作台总览 | 总览页 | 展示项目解析状态、源码概览、文档完成度、关键风险、最近文档和下一步入口，是进入模块地图、核心流程和验收契约的主入口。 |
| 阶段1(MVP) | 模块地图 | 模块地图页 | 展示 `code-back/mahjong` 的模块树、文件族分组、职责摘要、依赖关系和源码引用，帮助 lei 建立源码全局认知。 |
| 阶段1(MVP) | 核心流程 | 核心流程页 | 展示房间、用户、牌局、匹配、RPC、异常恢复等核心流程，包含流程图、涉及模块、关键状态、异常路径和源码引用。 |
| 阶段1(MVP) | 影响范围 | 影响范围页 | 展示需求类型、影响模块、影响流程、风险等级、验收点和推荐阅读，帮助 he 判断新需求影响范围。 |
| 阶段1(MVP) | 脚本索引 | 脚本索引页 | 展示轻量扫描/索引脚本的用途、输入输出、安全边界和最近运行结果，确保脚本只辅助解析、不修改业务代码。 |
| 阶段1(MVP) | 验收契约 | 验收契约页 | 展示 MVP 完成定义、AC 清单、门禁状态、未验证结论和进入 TDD master 的下一步入口。 |
| 阶段2 | 后续扩展 | 前端调试页 | 后续如需要，可基于解析结果增加麻将后端调试或管理界面。第一版只预留，不开发。 |
| 阶段3 | 后续扩展 | 代码实现页 | 后续进入真实代码实现和持续迭代时，用于展示任务、测试和交付状态。第一版只预留，不开发。 |

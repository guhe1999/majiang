# 电商调研 Playbook

## 适用场景

C 端电商平台、自营品牌网店、垂直电商、社交电商、直播电商、内容种草+转化

## 一、找竞品（5-8 个）

### 直接竞品（3-5 个）

**找法**：
- 京东 / 天猫 / 拼多多 同品类销量榜
- 抖音电商 / 视频号小店 同品类 TOP 主播
- 小红书电商 同品类品牌
- 独立站：用 Shopify Showcase / Built with Shopify 找海外参考
- 行业垂直平台：得物（潮流）/ 闲鱼（二手）/ Boss 直聘（招聘）等

**记录**：
- 品牌名 / 主营品类
- 一句话定位（性价比？品质？设计？速度？）
- 主营 SKU 数量
- 客单价区间
- 月销/年销（淘宝指数、生意参谋公开数据、品牌发布会数据）
- 主流量来源（推断）

### 间接竞品（2-3 个）

不卖同样东西但抢同样钱包/同样场景。

## 二、抓商品/店铺页

**用 WebFetch 抓**：
- 店铺/品牌主页（看主推、装修风格、活动）
- TOP 3 商品详情页（看主图、卖点、详情页结构、规格、价格、销量、评价数）
- 直播间链接（如果是直播电商）

**关键字段**：
- 主图前 5 张文案（暴露核心卖点排序）
- 商品标题关键词（暴露 SEO 重点）
- 详情页结构（场景图/对比图/工艺图/材质图/检测报告/明星代言/用户评价）
- 规格 SKU（颜色/尺码/型号/数量）
- 客单价 + 促销价 + 历史最低价（阿基米德历史价格查询）
- 销量数 + 评价数
- 累计客户数

## 三、挖差评

### 必查渠道

| 渠道 | 用法 |
|------|------|
| **京东评价** | 商品页 → "好评/中评/差评" 切换看差评 |
| **淘宝问大家** | 商品页 → "问大家" 看用户真实疑问 |
| **小红书** | 搜 `{品牌名} 真实测评` / `{品牌名} 踩雷` / `{品牌名} 退货` |
| **黑猫投诉** | tousu.sina.com.cn 搜品牌名 |
| **B 站** | 搜 `{品牌名} 测评` 看测评 UP 主 |
| **抖音** | 搜品牌名看用户视频 |

### 挖什么

- **质量投诉 Top 10**：货不对板、做工差、用料差、虚标
- **服务投诉 Top 5**：发货慢、客服差、退货难
- **物流投诉 Top 5**
- **价格投诉 Top 5**：先涨后降、双 11 比平时贵
- **真实使用场景**（看好评和真实测评里用户怎么用）

## 四、行业 Benchmark

### 必查指标

| 指标 | 健康范围 | 数据源 |
|------|---------|--------|
| **转化率** | 行业 1-3% (健康)，DTC 品牌 2-5% | 艾瑞 / 易观 / 国家统计局电商报告 |
| **客单价 AOV** | 因品类差异巨大 | 同上 |
| **复购率** | 美妆 30%+, 食品 40%+, 3C <10% | 友盟+ / 易观分析 |
| **退货率** | 服装 30%+, 美妆 5-10%, 3C 5% | 平台公开 |
| **加购转化** | 加购到下单 30-40% | 生意参谋 |
| **直播间停留时长** | 1 分钟+ (健康), 3 分钟+ (优秀) | 直播观研 / 蝉妈妈 |
| **直播间转化率** | 大主播 5-15%，腰部 1-3% | 同上 |
| **种草到转化路径长度** | 3-7 个内容触达 | 群邑 / 巨量算数 |

### 数据源

- **艾瑞咨询**（电商行业报告）
- **易观分析**
- **国家统计局**电商专题
- **生意参谋**（淘系，需账号）
- **蝉妈妈 / 飞瓜**（抖快直播数据）
- **千瓜数据**（小红书数据）
- **群邑 / 凯度**（消费者洞察）
- **Statista**（全球电商）
- **eMarketer**（全球电商）

## 五、关键判断辅助

### 这个电商生意值不值得做的检查清单

- [ ] 是否有差异化（同质化品类卷不动）
- [ ] 毛利能扛住流量成本（电商流量成本 30%+ 是常态）
- [ ] 供应链稳吗（自有工厂/OEM/经销）
- [ ] 复购模型清晰（消耗品最佳，耐用品要靠生态）
- [ ] 流量获取方式（自然搜索/付费投放/达人种草/直播/私域）
- [ ] 客服/物流能扛住峰值（大促爆单怎么办）
- [ ] 合规：食品/药品/医疗器械/化妆品/3C 各有特殊资质

### 中国电商特殊点

- 平台规则博弈：天猫规则 / 京东 PLUS / 抖音直播规则 / 小红书种草规则
- 大促波动巨大：618 / 双 11 / 双 12 / 三八 / 七夕等
- 物流成本：江浙沪发往全国 vs 仓库分布
- 真假货问题：拼多多/直播带货真假识别
- 中差评/差评师/恶意举报对接
- 内容种草是新基建：小红书+抖音+视频号矩阵
- 私域沉淀（企微/社群）是抗平台波动的关键

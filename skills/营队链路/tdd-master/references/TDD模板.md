## 5. 实现顺序建议（Coding Agent 回归证明协议）

> Coding Agent 按以下顺序实现，每条用例走红-绿-重构 + 回归证明四步：
> 1. 写测试 → 跑 → **确认失败**（失败方式与"预期失败原因"一致）
> 2. 写最小实现 → 跑 → **确认通过**
> 3. **回退实现** → 跑 → **确认再次失败**（证明不是假绿灯）
> 4. 恢复实现 → 跑 → **确认通过**

**P0 实现顺序**：`{{S-AUTH-01 → E-ORDER-01 → ...}}`

**P1 实现顺序**：`{{C-FORM-01 → B-PRICE-02 → ...}}`
